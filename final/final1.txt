numpy
-다차원의 배열 자료구조 클래스인 ndarray를 지원, 벡터와 행렬을 사용하는 선형대수 계산에 주로 사용
-다차원 배열을 처리하는데 필요한 여러 유용한 기능을 제공
pandas
-numpy 구조를 기반으로 데이터 분석을 위해 사용되는 라이브러리.
-데이터에 대한 다양한 연산 함수들을 제공
-다양한 외부 리소스의 데이터를 읽고 처리하는 기능 제공
matplotlib
-파이썬으로 데이터를 시각화할 때 가장 많이 사용하는 라이브러리
-2D 형태의 그래프, 이미지 등을 그릴 때 사용
-과학 컴퓨팅 연구 분야나 인공지능 연구 분야에서도 많이 활용
